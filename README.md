# Advent of Code 2024

## Setup

```
npm i
```

## Initiate a new day

```
npm run new-day DAY_NUMBER
```

## Run a specific day

### All day

```
npm run exec DAY_NUMBER
```

### Specific part

```
npm run exec DAY_NUMBER PART_NUMBER
```

### Benchmark a specific day

### All day

```
npm run benchmark DAY_NUMBER
```

### Specific part

```
npm run benchmark DAY_NUMBER PART_NUMBER
```
